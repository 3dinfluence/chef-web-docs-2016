The following options may be used with any of the arguments available to the |subcommand knife data bag| subcommand:

``--chef-zero-port PORT``
   |port chef_zero|

``-c CONFIG_FILE``, ``--config CONFIG_FILE``
   |config|

``-d``, ``--disable-editing``
   |no_editor|

``--defaults``
   |defaults|

``-e EDITOR``, ``--editor EDITOR``
   |editor|

``-E ENVIRONMENT``, ``--environment ENVIRONMENT``
   |name environment| When this option is added to a command, the command will run only against the named environment.

``-F FORMAT``, ``--format FORMAT``
   |format|

``-h``, ``--help``
   |help subcommand|

``-k KEY``, ``--key KEY``
   |key|

``--[no-]color``
   |color|

``--print-after``
   |print_after|

``-s URL``, ``--server-url URL``
   |url chef_server|

``-u USER``, ``--user USER``
   |user knife|

``-V``, ``--verbose``
  |verbose|

``-v``, ``--version``
   |version chef|

``-y``, ``--yes``
   |yes|

``-z``, ``--local-mode``
   |local_mode|
