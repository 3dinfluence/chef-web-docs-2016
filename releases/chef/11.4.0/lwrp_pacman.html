pacman Lightweight Resource¶

The pacman cookbook is used to install the pacman package manager and configure groups.

The pacman cookbook contains the following lightweight resources: pacman_aur and pacman_group.

Note

These lightweight resources are part of the pacman cookbook (http://community.opscode.com/cookbooks/pacman).

pacman_aur¶

The pacman_aur lightweight resource is used to use the Arch Linux package manager (pacman). A custom package description file (PKGBUILD) can be used when creating packages.

For more information, see: https://wiki.archlinux.org/index.php/PKGBUILD.

Actions¶

This lightweight resource provider has the following actions:

Action Description
:build Use to build a package using the information stored in a PKGBUILD script.
:install Use to install a package that has been built with the :build action.

Attributes¶

This lightweight resource provider has the following actions:

Action Description
builddir The directory to which a package is downloaded. Default value: Chef::Config[:file_cache_path]/builds.
exists Indicates that a package already exists.
options One (or more) additional options that are passed to the command.
package_name The name of the package.
patches An array of files to be used as patches for the package. Default value: [].
pkgbuild_src Indicates that a custom PKGBUILD file will be used. When true, the pacman_aur lightweight resource will look for a cookbook_file named PKGBUILD to use as package instructions. Default value: false.
version The version of a package to be installed or upgraded.

Examples¶

To use a simple AUR package

pacman_aur "djbdns" do
  action [:build, :install]
end

To use a custom package with PKGBUILD

pacman_aur "netatalk" do
  pkgbuild_src "PKGBUILD"
  action [:build,:install]
end

pacman_group¶

The pacman_group lightweight resource is used to manage a pacman package group.

Actions¶

This lightweight resource provider has the following actions:

Action Description
:install Use to install a group.
:remove Use to remove a group.

Attributes¶

This lightweight resource provider has the following attributes:

Attribute Description
options One (or more) additional options that are passed to the command.
package_name The name of the package group.

Examples¶

To remove the “base-devel” group:

pacman_group "base-devel" do
  action :remove
end
