rabbitmq Lightweight Resource¶

The rabbitmq cookbook is used to install and configure RabbitMQ.

The rabbitmq cookbook contains the following lightweight resources: rabbitmq_plugin, rabbitmq_user, and rabbitmq_vhost.

Note

These lightweight resources are part of the rabbitmq cookbook (http://community.opscode.com/cookbooks/rabbitmq).

rabbitmq_plugin¶

The rabbitmq_plugin lightweight resource is used to enable or disable a RabbitMQ plugin.

Actions¶

This lightweight resource provider has the following actions:

Action Description
:disable Use to disable a plugin.
:enable Default. Use to enable a plugin.

Attributes¶

This lightweight resource provider has the following attributes:

Attribute Description
plugin The name of the plugin.

Examples¶

To enable a plugin:

rabbitmq_plugin "rabbitmq_stomp" do
  action :enable
end

To disable a plugin:

rabbitmq_plugin "rabbitmq_shovel" do
  action :disable
end

rabbitmq_user¶

The rabbitmq_user lightweight resource is used to add and delete users.

Actions¶

This lightweight resource provider has the following actions:

Action Description
:add Use to add a user (with a password).
:clear_permissions Use to clear permissions for a user.
:delete Use to delete a user.
:set_permissions Use to set permissions for a user. (The virtual host is optional.)

Attributes¶

This lightweight resource provider has the following attributes:

Attribute Description
password The user password.
permissions One (or more) permissions that will be set.
user The name of the user.
vhost The name of the virtual host.

Examples¶

To add a user:

rabbitmq_user "nova" do
  password "sekret"
  action :add
end

To delete a user:

ruby rabbitmq_user "guest" do
  action :delete
end

To set the permissions for a user:

rabbitmq_user "nova" do
  vhost "/nova"
  permissions "\".\" \".\" \".*\""
  action :set_permissions
end

rabbitmq_vhost¶

The rabbitmq_vhost lightweight resource is used to add and delete virtual hosts.

Actions¶

This lightweight resource provider has the following actions:

Action Description
:add Use to add a virtual host.
:delete Use to delete a virtual host.

Attributes¶

This lightweight resource provider has the following attributes:

Attribute Description
vhost The name of the virtual host.

Examples¶

To add a virtual host:

ruby rabbitmq_vhost "/nova" do
  action :add
end
