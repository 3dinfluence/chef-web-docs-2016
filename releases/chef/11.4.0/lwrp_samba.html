samba Lightweight Resource¶

The samba cookbook is used to install and configure Samba.

The samba cookbook contains the following lightweight resource: samba_user.

samba_user¶

The samba_user lightweight resource lightweight resource is used to manage password entries for Samba.

Note

This lightweight resource is part of the samba cookbook (http://community.opscode.com/cookbooks/samba).

Actions¶

This lightweight resource provider has the following actions:

Action Description
:create Use to create a user.
:enable Use to enable a user.
:delete Use to enable a user.

Attributes¶

This lightweight resource provider has the following attributes:

Attribute Description
password The user password. This is passed to the smbpasswd program in Samba.

Examples¶

To create a user:

samba_user "jtimberman" do
  password "SuperSecret"
  action :create
end
