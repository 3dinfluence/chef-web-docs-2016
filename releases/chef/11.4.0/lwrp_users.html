users Lightweight Resource¶

The users cookbook is used to create users based on a data bag search.

The users cookbook contains the following lightweight resource: users_manage.

users_manage¶

The users_manage lightweight resource is used to add and remove users from security groups, based on the presence of group attributes for user objects in a data bag.

Note

This lightweight resource is part of the users cookbook (http://community.opscode.com/cookbooks/users).

Actions¶

This lightweight resource provider has the following actions:

Action Description
:create Use to create a user.
:remove Use to remove a user.

Attributes¶

This lightweight resource provider has the following attributes:

Attribute Description
cookbook The name of the cookbook in which the authorized_keys template is located.
data_bag The object to be searched.
group_id The identifier for the group.
group_name The name of the group. This value defaults to the name of the resource if left empty.
search_group The name of the group to be searched. This value defaults to the name of the resource if left empty.

Examples¶

To create users based on data bag objects that have a group_id attribute value of 2300:

users_manage "sysadmin" do
  group_id 2300
  action :create
end

To remove users based on data bag objects that have a group_id attribute value of 2300:

users_manage "sysadmin" do
  group_id 2300
  action :remove
end
