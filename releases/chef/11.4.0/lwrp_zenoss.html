zenoss Lightweight Resource¶

The zenoss cookbook is used to install and configure Zenoss, and then register nodes as devices.

The zenoss cookbook contains the following lightweight resources: zenoss_zenbatchload, zenoss_zenmd, zenoss_zenpack, and zenoss_zenpatch.

Note

These lightweight resources are part of the zenoss cookbook (http://community.opscode.com/cookbooks/zenoss).

zenoss_zenbatchload¶

The zenoss_zenbatchload lightweight resource is used to populate a Zenoss server based on a list of devices, their device classes, systems, groups, locations, and properties.

Actions¶

This lightweight resource provider has the following actions:

Action Description
:run  

Attributes¶

This lightweight resource provider has the following attributes:

Attribute Description
devices  
groups  
locations  
systems  

Examples¶

None.

zenoss_zenmd¶

The zenoss_zenmd lightweight resource is used to pass a command to the ZenDMD command-line utility on a Zenoss server for execution.

Actions¶

This lightweight resource provider has the following actions:

Action Description
:deviceclass  
:group  
:location  
:run  
:system  
:users  

Attributes¶

This lightweight resource provider has the following attributes:

Attribute Description
address  
command  
description  
location  
modular_plugins  
name  
properties  
templates  
users  

Examples¶

None.

zenoss_zenpack¶

A ZenPack is used to add or override functionality within the Zenoss platform. The zenoss_zenpack lightweight resource is used to install a ZenPack on a Zenoss server.

Actions¶

This lightweight resource provider has the following actions:

Action Description
:install Use to install a ZenPack.
:remove Use to remove a ZenPack.

Attributes¶

This lightweight resource provider has the following attributes:

Attribute Description
exists  
package  
py_version  
version  

Examples¶

None.

zenoss_zenpatch¶

The zenoss_zenpatch lightweight resource is used to install patches on a Zenoss server.

Actions¶

This lightweight resource provider has the following actions:

Action Description
:install Use to install a patch on a Zenoss server.

Attributes¶

This lightweight resource provider has the following attributes:

Attribute Description
svnpatch  
ticket  

Examples¶

None.
