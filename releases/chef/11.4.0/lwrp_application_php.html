application_php Lightweight Resource¶

Warning

DRAFT

The application_php cookbook is used to PHP applications.

The application_php cookbook contains the following lightweight resources: application_php and application_mod_php.

Note

These lightweight resources are part of the application_php cookbook (http://community.opscode.com/cookbooks/application_php).

application_php¶

The application_php_php lightweight resource is used to deploy PHP web applications from a source control management repository. This lightweight resource uses the deploy_revision provider from the deploy resource to perform the bulk of its tasks.

Actions¶

This lightweight resource provider has the following actions:

Action Description
:deploy Use to deploy an application.
:force_deploy Use to remove any existing release of the same code version and re-deploy a new one in its place.

Attributes¶

This lightweight resource provider has the following attributes:

Attribute Description
database_master_role The name of a role. During a Chef run, a search will be performed to find a node with the same name as this value in the same environment as the current role. If a node is found, it’s IP address will be used when rendering the context file.
local_settings_file The name of the local settings file to be generated by a template. Default value: LocalSettings.php.
packages An array of PEAR packages to install.
settings_template The name of a template that will be rendered to create the local settings file. Default value: #{local_settings_file}.erb.

Examples¶

None.

application_mod_php¶

The application_php_mod_php_apache2 lightweight resource is used to configure Apache and mod_php to run the application by creating a virtual host.

Actions¶

This lightweight resource provider has the following actions:

Action Description
:deploy Use to deploy an application.
:force_deploy Use to remove any existing release of the same code version and re-deploy a new one in its place.

Attributes¶

This lightweight resource provider has the following attributes:

Attribute Description
server_aliases An array of server aliases.
webapp_template The template used to render the virtual host configuration file. Default value: php.conf.erb.

Examples¶

None.
