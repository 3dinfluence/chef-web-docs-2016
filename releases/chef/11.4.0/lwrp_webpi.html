webpi Lightweight Resource¶

The webpi cookbook is used to run the Microsoft Web Platform Installer (WebPI).

The webpi cookbook contains the following lightweight resource: webpi_product.

webpi_product¶

The webpi_product lightweight resource is used to automate and service installations for Windows Azure web roles using WebPlCmdLine.exe. For more information about WebPlCmdLine.exe, see: http://msdn.microsoft.com/en-us/library/windowsazure/gg433092.aspx.

Note

This lightweight resource is part of the webpi cookbook (http://community.opscode.com/cookbooks/webpi).

Actions¶

This lightweight resource provider has the following actions:

Action Description
:install Use to install a product.

Attributes¶

This lightweight resource provider has the following attributes:

Attribute Description
accept_eula Indicates that an end-user license agreement will be accepted automatically. Default value: false.
product_id The identifier of the product to be installed.

Examples¶

To install Windows PowerShell 2.0:

webpi_product "PowerShell2" do
  accept_eula true
  action :install
end

To install the recommended configuration for Internet Information Services (IIS) 7:

webpi_product "IIS7" do
  accept_eula true
  action :install
end
