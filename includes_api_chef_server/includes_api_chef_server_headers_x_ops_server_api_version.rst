Use ``X-Ops-Server-API-Version`` to specify the version of the |api chef server|. For example: ``X-Ops-Server-API-Version: 1``. ``X-Ops-Server-API-Version: 0`` is supported for use with the version 12 |chef server|, but will be deprecated as part of the next major release.
