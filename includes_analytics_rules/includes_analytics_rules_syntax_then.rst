Use the ``then`` statement to specify what to do when a rule in the ``when`` statement is matched. For example:

.. code-block:: ruby

   rule on action when
     data.name="app1"
   then
     set(#foo, "100")  
   end

When the name matches ``app1``, ``#foo`` will be set to ``100``.
