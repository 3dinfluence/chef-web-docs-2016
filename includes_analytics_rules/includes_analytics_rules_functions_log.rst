Use ``log`` to log a message via the configured logger as a string.

The syntax for this function:

.. code-block:: java

   log(messageToLog)

where:

* ``messageToLog`` is the message that is added to the log

For example:

.. code-block:: java

   log("Added rule for org <obj.organization_name>")
