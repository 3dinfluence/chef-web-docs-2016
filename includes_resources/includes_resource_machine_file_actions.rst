This resource has the following actions:

.. list-table::
   :widths: 150 450
   :header-rows: 1

   * - Action
     - Description
   * - ``:delete``
     - Use to delete xxxxx from a machine.
   * - ``:download``
     - Use to download xxxxx from a machine.
   * - ``:nothing``
     - Use to do nothing.
   * - ``:upload``
     - Default. Use to upload xxxxx to a machine.
