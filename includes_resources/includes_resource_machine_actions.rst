This resource has the following actions:

.. list-table::
   :widths: 150 450
   :header-rows: 1

   * - Action
     - Description
   * - ``:converge``
     - Use to converge a machine.
   * - ``:create``
     - Default. Use to create a machine.
   * - ``:destroy``
     - Use to destroy a machine.
   * - ``:nothing``
     - Use to do nothing.
   * - ``:stop``
     - Use to stop a machine.
