Use the template resource when a template is stored a cookbook's ``/templates`` directory:

.. code-block:: python
       
   motd
      ├── Berksfile
      ├── README.md
      ├── chefignore
      ├── metadata.rb
      ├── recipes
      │   └── default.rb
      └── templates
          └── default
              └── motd.erb

This template is then rendered to create a file:

.. code-block:: ruby
