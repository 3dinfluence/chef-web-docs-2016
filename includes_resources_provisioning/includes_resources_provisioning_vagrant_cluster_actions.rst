The ``vagrant_cluster`` driver-specific resource has the following actions:

.. list-table::
   :widths: 150 450
   :header-rows: 1

   * - Action
     - Description
   * - ``:create``
     - Default. Use to create a cluster of machines in |vagrant|.
   * - ``:delete``
     - Use to delete a cluster of machines in |vagrant|.
   * - ``:nothing``
     - Use to do nothing.
