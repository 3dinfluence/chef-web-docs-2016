The ``fog_key_pair`` driver-specific resource has the following actions:

.. list-table::
   :widths: 150 450
   :header-rows: 1

   * - Action
     - Description
   * - ``:create``
     - Default. Use to create a key pair.
   * - ``:delete``
     - Use to destroy a key pair.
   * - ``:nothing``
     - Use to do nothing.
