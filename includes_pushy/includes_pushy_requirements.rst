|pushy| has the following requirements:

* |chef server oec| version 11.0.1 (or later)
* TCP protocol ports 10000-10003

The |pushy| client can run on the following platforms:

* |ubuntu| 10.04, 10.10, 12.04, and 12.10
* |centos| 5 and 6
* |windows|
