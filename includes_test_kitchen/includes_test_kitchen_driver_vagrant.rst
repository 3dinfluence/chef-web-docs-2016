The ``kitchen-vagrant`` driver for |kitchen| generates a single ``vagrantfile`` for each instance of |kitchen| in a sandboxed directory. The ``kitchen-vagrant`` driver supports |virtualbox| and |vmware fusion| and requires |vagrant| 1.1.0 (or higher).
