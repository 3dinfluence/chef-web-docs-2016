|search special character| The following characters can be included within the search query syntax, but each occurrence of a special character must be escaped with a backslash (``\``)::

   +  -  &&  | |  !  ( )  { }  [ ]  ^  "  ~  *  ?  :  \

For example::

   \(1\+1\)\:2
