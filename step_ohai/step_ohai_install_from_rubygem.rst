To install |ohai| from |rubygems|, do the following:

#. Add the |gem| server to the local resources:

   .. code-block:: bash

      $ sudo gem source -a http://rubygems.org

#. Install |ohai|:

   .. code-block:: bash

      $ sudo gem install ohai
