Installs a reference database configuration for use with |openstack grizzly| and |chef openstack|. This cookbook is not required and alternate database configurations may be used. Supported databases: |mysql| (and will support |postgresql| soon).

This cookbook is located at: https://github.com/stackforge/cookbook-openstack-ops-database.
