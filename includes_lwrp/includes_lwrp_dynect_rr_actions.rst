This lightweight resource provider has the following actions:

.. list-table::
   :widths: 200 300
   :header-rows: 1

   * - Action
     - Description
   * - ``:create``
     - Use to create a record.
   * - ``:delete``
     - Use to delete a record.
   * - ``:replace``
     - Use to replace an existing record.
   * - ``:update``
     - Use to update an existing record.
