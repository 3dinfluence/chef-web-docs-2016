This lightweight resource provider has the following actions:

.. list-table::
   :widths: 200 300
   :header-rows: 1

   * - Action
     - Description
   * - ``:allow``
     - Use to set a rule that allows incoming traffic.
   * - ``:deny``
     - Use to set a rule that denies incoming traffic.
   * - ``:reject``
     - Use to set a rule that rejects incoming traffic.
