This lightweight resource provider has the following actions:

.. list-table::
   :widths: 200 300
   :header-rows: 1

   * - Action
     - Description
   * - ``:create``
     - Default. Use to create a |yum| repository file and build a |yum| repository listing.
   * - ``:delete``
     - Use to delete a |yum| repository file.
