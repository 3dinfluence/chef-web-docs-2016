This lightweight resource provider has the following attributes:

.. list-table::
   :widths: 200 300
   :header-rows: 1

   * - Attribute
     - Description
   * - ``application``
     - The application or site to which a module will be added.
   * - ``module_name``
     - The name of the module.
   * - ``type``
     - The module type.
   * - ``precondition``
     - The pre-condition for a module, if applicable.
