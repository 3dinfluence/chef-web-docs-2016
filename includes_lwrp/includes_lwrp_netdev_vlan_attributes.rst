This lightweight resource provider has the following attributes:

.. list-table::
   :widths: 200 300
   :header-rows: 1

   * - Attribute
     - Description
   * - ``description``
     - The description of the VLAN.
   * - ``name``
     - The name of the VLAN.
   * - ``vlan_id``
     - The unique identifier for the VLAN.
