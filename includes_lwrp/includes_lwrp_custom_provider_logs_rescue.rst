Use the ``rescue`` clause to make sure that a log message is always provided. For example:

.. code-block:: ruby

   def load_current_resource
     ...
     begin
       ...
     rescue
       Chef::Log.debug("Cannot find #{@new_resource} in the swarm")
     end
     ...
   end
