Extract the ``chef-starter.zip`` file to a directory on the workstation. Open a command prompt and change into the |chef repo| directory extracted from the starter kit. For example:

.. code-block:: bash

   $ cd ~/Downloads
   $ unzip chef-starter.zip
   $ cd chef-repo
