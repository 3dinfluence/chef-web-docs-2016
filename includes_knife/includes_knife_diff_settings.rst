The following ``knife diff`` settings can be added to the |knife rb| file:

``knife[:name_only]``
   Use to add the ``--name-only`` option.

``knife[:name_status]``
   Use to add the ``--name-status`` option.

``knife[:recurse]``
   Use to add the ``--recurse`` option.
