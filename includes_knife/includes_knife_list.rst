Use the |subcommand knife list| subcommand to view a list of objects on the |chef server|. This subcommand works similar to ``knife cookbook list``, ``knife data bag list``, ``knife environment list``, ``knife node list``, and ``knife role list``, but with a single verb (and a single action).
