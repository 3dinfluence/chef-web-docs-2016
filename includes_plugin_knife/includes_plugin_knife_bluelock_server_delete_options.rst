This argument has the following options:

``-A USER_NAME``, ``--bluelock-username USER_NAME``
   |name user bluelock|

``-K PASSWORD``, ``--bluelock-password PASSWORD``
   |password bluelock|

``-N NODE_NAME``, ``--node-name NODE_NAME``
   |node_purge| This must be used with the ``--purge`` option.

``-p``, ``--purge``
   |purge node bluelock|
