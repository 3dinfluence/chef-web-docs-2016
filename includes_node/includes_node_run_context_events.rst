.. notes for future documentation: The ``events`` object is used to generate logs and create reports during a |chef client| run. Events are a simple, queue-free publish and subscribe model. Log formatters---``:min``, ``:doc``, and so on---are examples of event dispatcher subclasses of the ``events`` object.
