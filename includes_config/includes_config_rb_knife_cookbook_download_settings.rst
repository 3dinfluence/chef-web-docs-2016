The following ``knife cookbook download`` settings can be added to the |knife rb| file:

``knife[:download_directory]``
   Adds the the ``--dir`` option.

``knife[:force]``
   Adds the the ``--force`` option.

``knife[:latest]``
   Adds the the ``--latest`` option.
