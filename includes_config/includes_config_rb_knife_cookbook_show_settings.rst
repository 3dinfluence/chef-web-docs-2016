The following ``knife cookbook show`` settings can be added to the |knife rb| file:

``knife[:fqdn]``
   Adds the the ``--fqdn`` option.

``knife[:platform]``
   Adds the the ``--platform`` option.

``knife[:platform_version]``
   Adds the the ``--platform-version`` option.
