The following ``knife cookbook delete`` settings can be added to the |knife rb| file:

``knife[:all]``
   Adds the the ``--all`` option.

``knife[:print_after]``
   Adds the the ``--print-after`` option.

``knife[:purge]``
   Adds the the ``--purge`` option.
