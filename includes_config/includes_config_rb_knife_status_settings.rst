The following ``knife status`` settings can be added to the |knife rb| file:

``knife[:hide_healthy]``
   Adds the the ``--hide-healthy`` option.

``knife[:run_list]``
   Adds the the ``--run-list`` option.

``knife[:sort_reverse]``
   Adds the the ``--sort-reverse`` option.
