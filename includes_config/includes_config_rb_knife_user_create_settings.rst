The following ``knife user create`` settings can be added to the |knife rb| file:

``knife[:admin]``
   Adds the the ``--admin`` option.

``knife[:file]``
   Adds the the ``--file`` option.

``knife[:user_key]``
   Adds the the ``--user-key`` option.

``knife[:user_password]``
   Adds the the ``--password`` option.
