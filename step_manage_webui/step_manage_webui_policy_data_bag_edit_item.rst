To edit a data bag item:

#. Open |webui|.
#. Click |webui policy|.
#. Click |webui policy_data_bags|.
#. Select a data bag.
#. Select the |webui tab_items| tab.
#. Select a data bag.
#. Click **Edit**.

   .. image:: ../../images/step_manage_webui_policy_data_bag_edit_item.png

#. Make your changes.
#. Click |webui button_save_data_bag_item|.
