To edit a data bag item:

#. Open the |chef manage|.
#. Click **Policy**.
#. Click **Data Bags**.
#. Select a data bag.
#. Select the **Items** tab.
#. Select a data bag.
#. Click **Edit**.

   .. image:: ../../images/step_manage_webui_policy_data_bag_edit_item.png

#. Make your changes.
#. Click **Save Item**.
