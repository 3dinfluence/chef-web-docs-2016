To edit node attributes:

#. Open the |chef manage|.
#. Click **Nodes**.
#. Select a node.
#. Click the **Attributes** tab.
#. Click **Edit**.
#. In the **Edit Node Attributes** dialog box, make your changes:

   .. image:: ../../images/step_manage_webui_node_attributes_edit.png

#. Click **Save Attributes**.
