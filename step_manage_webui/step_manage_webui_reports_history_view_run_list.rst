To view |chef client| run-list details:

#. Open the |chef manage|.
#. Click **Reports**.
#. Click **Run History**.
#. Select the range of |chef client| runs to show, the correct environment, and correct status.
#. Select a |chef client| run.
#. Select the **Run List** tab:
