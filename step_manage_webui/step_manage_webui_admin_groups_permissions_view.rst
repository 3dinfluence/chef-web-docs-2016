To view group permissions:

#. Open the |chef manage|.
#. Click **Administration**.
#. Click **Groups**.
#. Select a group.
#. Click the **Group Permissions** tab.
#. Set the appropriate permissions: **Delete**, **Grant**, **Read**, and/or **Update**.
