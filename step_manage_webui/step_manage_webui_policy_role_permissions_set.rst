To set permissions list for a role object:

#. Open the |chef manage|.
#. Click **Policy**.
#. Click **Roles**.
#. Select a role.
#. Click the **Permissions** tab.
#. For each group listed under **Name**, select or de-select the **Read**, **Update**, **Delete**, and **Grant** permissions.
