.. This is an included how-to. 


To set tags for a node (or a group of nodes):

#. Open |opscode manage|.
#. Click |opscode manage nodes|.
#. Select a node (or a group of nodes).
#. Click |opscode manage nodes_manage_tags|.
#. In the |opscode manage db_manage_node_tags| dialog box, enter the name of the tag and then select **Set Tags** from the drop-down.

   .. image:: ../../images/step_manage_webui_node_tags_set.png

#. Click |opscode manage button_update_tags|.
