.. This is an included how-to. 


To filter report histories by |chef client| run status:

#. Log on to |opscode manage|.
#. Click |opscode manage reports|.
#. Click |opscode manage reports_run_history|.
#. From the |opscode manage button_filter_by_status| drop-down, select ``All``, ``Success``, ``Failure``, or ``Started``:
