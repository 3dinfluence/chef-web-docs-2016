.. This is an included how-to. 


To view all of the nodes:

#. Open |opscode manage|.
#. Click |opscode manage nodes|.

   where:

   .. list-table::
      :widths: 60 420
      :header-rows: 1
   
      * - Setting
        - Description
      * - ``Node Name``
        - |name node|
      * - ``Platform``
        - |platform_node|
      * - ``FQDN``
        - |node_fqdn|
      * - ``IP Address``
        - |ip_address node|
      * - ``Uptime``
        - |node_uptime|
      * - ``Last Check-in``
        - |node_last_checkin|
      * - ``Environment``
        - |node_environment|
      * - ``Actions``
        - The |opscode manage| web user interface tasks available for this node.
