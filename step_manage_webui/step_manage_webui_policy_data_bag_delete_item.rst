To delete a data bag item:

#. Open **Chef Manage**.
#. Click |webui policy|.
#. Click |webui policy_data_bags|.
#. Select a data bag.
#. Select the |webui tab_items| tab.
#. Select a data bag.
#. Click **Delete**.
