To delete a data bag item:

#. Open |opscode manage|.
#. Click |opscode manage policy|.
#. Click |opscode manage policy_data_bags|.
#. Select a data bag.
#. Select the |opscode manage tab_items| tab.
#. Select a data bag.
#. Click **Delete**.
