To view a cookbook file:

#. Open the |chef manage|.
#. Click **Policy**.
#. Click **Cookbooks**.
#. Select a cookbook.
#. Click the **Content** tab.
#. Select the file type: **Attributes**, **Definitions**, **Files**, **Recipes**, **Templates**, or **Root Files**.
